## Setting up for dev:

update pip and install uv 
```bash
pip install --upgrade pip
pip install uv
```

Sync project
```bash
cd ml
source .venv/bin/activate
uv sync
```

## Running the API

Make sure you 

uvicorn canonical launch:

```bash
uv run uvicorn ml.main:app --host 0.0.0.0 --port 8000 u
```
